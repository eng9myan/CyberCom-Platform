POS Session User Restrict

Control and restrict user access to specific Point of Sale sessions.

  • Restrict POS session access to selected users only
  • Prevent unauthorized users from opening POS sessions
  • Enhances security in multi-user and multi-counter environments
  • Easy configuration from POS settings
  • Fully compatible with standard Cycom POS workflows

Step 1

Navigate to the Point of Sale module.

Step 2

Go to Configuration and select Point of Sale.

Configure Allowed Users

Add users in the POS Allowed Accessible Users field. Only the selected users will be able to access this POS session. If no users are selected, the session will not be accessible to anyone.

Session-Based User Access

Users are assigned per POS configuration, ensuring controlled access to individual POS sessions.

Restricted POS Access

Only authorized users can log in and access the assigned POS session. Unauthorized users are automatically restricted.
