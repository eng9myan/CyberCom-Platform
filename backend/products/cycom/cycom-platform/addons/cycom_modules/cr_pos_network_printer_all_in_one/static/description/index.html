Configuration Guide

Follow the steps below to configure & start printing from Cycom POS.

🖥️ Open POS Settings → Connected Devices
  • Navigate to the Point of Sale menu.
  • Click Settings.
  • Search for the Connected Devices section.
🖨️ Enable the ePos Printer Option
  • Enable the ePos Printer toggle in Connected Devices.
  • A printer selection dropdown will appear once enabled.
  • Select the printer you want to use for printing POS order receipts.
🍳 Enable Kitchen / Preparation Printing
  • Go to your POS Settings.
  • Search for the Preparation option.
  • Enable it to activate kitchen receipt printing.
➕ Add or Manage a Preparation Printer
  • Enable the Preparation Printer option.
  • Click Add Printer to add a new preparation printer.
  • Or click Manage Printers to update an existing one.
⚙️ Assign a Custom Printer for Preparation
  • Go to Manage Printers.
  • Select the preparation printer you created.
  • In Printer Type, enable Use a Custom Printer.
  • From the dropdown, select the specific printer you wish to use.
▶️ Start a POS Session & Print Automatically
  • Open a POS Session and create an order.
  • Use the Send option to manually send kitchen receipts.
  • If auto receipt printing is enabled, receipts will print automatically.
