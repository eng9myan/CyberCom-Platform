## Key Legend
*   **Yes:** Fully supported out-of-the-box natively.
*   **Basic:** Minimal functionality (e.g., standard posting, lacking complex currency derivatives or forecasting).
*   **Mod / Advanced:** Decades of specialized industry-vertical templates and automated tools.
*   **No:** Not currently supported; scheduled for future roadmap phases.
