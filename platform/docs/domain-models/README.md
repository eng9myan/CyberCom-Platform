# Domain Models

> Bounded contexts, aggregates, and ubiquitous language per business domain — DDD-aligned.

This folder was created during **Program 0, Phase 0.2 (Governance)**.
Content will be authored progressively under the governance defined in [../../CONTRIBUTING.md](../../CONTRIBUTING.md) and [../governance/repository_operating_model.md](../governance/repository_operating_model.md).

## Scope

- **Bounded contexts** per domain (Healthcare, Finance, HR, Identity, Citizen, …).
- **Aggregates, entities, value objects** with invariants.
- **Ubiquitous language glossary** per context.
- **Context maps** showing relationships (Shared Kernel, Customer/Supplier, ACL, OHS).

## Conventions

- One folder per bounded context once authored.
- Diagrams in Mermaid or PlantUML.
- Glossary terms are normative — code, docs, and APIs must use them.
